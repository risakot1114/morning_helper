# 🌅 Morning Helper（モーニングヘルパー）

## 📘 アプリケーション概要

- **アプリ名**: Morning Helper（モーニングヘルパー）
- **概要**: 朝の支度を楽しく・スムーズにするために、天気・服装・持ち物・花粉・占い情報をひと目で確認できるモバイルファーストWebアプリです。
- **できること**:
  - 現在地の天気・気温・降水確率などの表示
  - 気温や天気に応じた「ビジネス/カジュアル/キッズ」別の服装提案
  - その日の外出に役立つ持ち物アドバイス
  - 花粉情報（レベル・種類・3日予報）
  - 12星座占い（総合運/恋愛運/仕事運/健康運、ラッキーアイテム等）
  - 週間天気・週間コーデの専用ページ

## 🚶 利用方法

1. アクセス後、位置情報の利用を許可します（GPSで自動取得）
2. ホームで「現在の天気」「きょうのコーデ案」「おすすめの持ち物」「花粉」「占い」を確認
3. 服装スタイルを「ビジネス/カジュアル/キッズ」から選択して提案を切り替え
4. フッターの「週間」タブから、週間天気と週間コーデを確認

## 🎯 アプリケーションを作成した背景

朝は「時間がない」「情報が散らばっている」という課題があり、天気・服装・持ち物・占いなど“朝に必要な情報”を1つに集約することで、迷わず楽しく準備を進められる体験を目指しました。忙しいビジネスパーソンや子育て世帯にも直感的に使えるUIを重視しています。

## ✨ 工夫したポイント

- **モバイルファーストの設計**: スマートフォンでの操作を最優先に考え、朝の忙しい時間でも使いやすい大きめのボタンと文字サイズにしました。
- **デザインの親しみやすさ**: 雲の背景やパステルカラーを使って、朝の準備を楽しく感じられるデザインにしました。
- **服装提案の細かい調整**: 気温だけでなく季節感も考慮して、同じ気温でも秋と春で異なる服装を提案できるようにしました。
- **ページ表示の高速化**: 天気情報や占いデータを一度取得したら、30分〜2時間は保存しておくことで、同じデータを何度もAPIに取りに行かずに済むようにしました。これによりページ表示が速くなります。
- **情報の整理**: ホームページと週間ページを分けることで、必要な情報を探しやすくしました。

## 🔧 改善点

- **ログイン機能の実装**: ユーザーごとに好みのスタイル（ビジネス/カジュアル/キッズ）を保存できるようにする。
- **デザインの統一**: アイコンやボタンのデザインをより統一感のあるものにする。
- **エラーメッセージの改善**: APIエラー時に、ユーザーにわかりやすいメッセージを表示する。
